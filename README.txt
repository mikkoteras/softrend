A very brief introduction:

command line args:

-w <window-width>
-h <window-height>
-o <path-to-obj-file>

mouse and keyboard controls:

Ctrl + q: exit
Ctrl + l: toggle lights (between 100% ambient and 20% ambient + 100% directional)
Ctrl + c: toggle coordinate system display
Ctrl + i: toggle info text overlay
Ctrl + n: toggle normal vector display
Ctrl + p: toggle Phong reflection vector display
Ctrl + s: rotate shading model
Ctrl + r: reset scene
w,a,s,d: move viewer position around the object
q,e: tilt viewer position
f,r: move viewer closer and further from the object
Left mouse button + drag: move viewer position around the object
Mouse scroll wheel: move viewer position around the object
